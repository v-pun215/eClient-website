Introducing eClient, a Minecraft launcher with tons of features.

Learn more, at GitHub.

Features

  • Can launch all versions of Minecraft Java Edition (starting from 1.0).
  • Has a Mod Installer that I fixed (from PyCraft).
  • Can install Forge and Fabric.
  • Ability to login with Mojang, Ely.by and Offline Accounts.
  • Never gives you up.
